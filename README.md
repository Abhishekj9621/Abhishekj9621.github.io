# Abhishek Jaiswal — Portfolio

A premium personal portfolio website built with React, featuring:
- Dark editorial UI with animated particle canvas hero
- Typewriter role animation
- Tabbed experience section
- Filterable project case studies with modal details
- Expandable skill group cards with proficiency bars
- Alternating achievement timeline
- Functional contact form
- Smooth scroll reveal animations throughout
- Fully responsive across all screen sizes

## Tech Stack
- React 18
- react-icons for all iconography
- CSS Custom Properties for the design system
- No external UI library (100% custom)

## Getting Started

```bash
npm install
npm start
```

## Build for Production

```bash
npm run build
```

The `build/` folder contains the static files ready to deploy to any static hosting (Vercel, Netlify, GitHub Pages, etc.).

## Deployment (Vercel — recommended)
1. Push to GitHub
2. Import repo in Vercel
3. Deploy — zero config needed

## Design System
- Font Display: Syne (headings)
- Font Body: DM Sans (body text)
- Font Mono: JetBrains Mono (labels, code)
- Primary accent: #00D4FF (Cyan)
- Secondary accent: #7C3AFF (Violet)
- Tertiary: #00E5A0 (Emerald)
- Background: #050A14
